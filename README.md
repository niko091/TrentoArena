# TrentoArena

## Install
1) ```npm ci```

## Setup
1) create ```.env``` from repository's secret
2) use ```/admin``` page to populate the db (user: ```admin```, pwd: ```admin123```)

## Build
1) ```npm run build```

## Run
1) ```npm start```

## Test
1) ```npm test```
